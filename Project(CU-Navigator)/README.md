## 🌐 Live Demo
👉https://cu-navigator.netlify.app/

---

## 📌 Description
This project implements **Dijkstra’s Algorithm** to find the shortest path between locations on a campus map.

It provides an interactive UI where users can:
- Select start and destination points
- Visualize shortest path
- View distance and estimated time

---

## 🛠️ Tech Stack
- JavaScript (ES Modules)
- Leaflet.js (Map rendering)
- Vite (Build tool)
- Dijkstra Algorithm (Core logic)
